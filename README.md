> Simple TBW & Profit Share Calculator for PHP.

## Installation

Require this package, with [Composer](https://getcomposer.org/), in the root directory of your project.

```bash
$ composer require arkx/ark-calculus
```

## Testing

``` bash
$ phpunit
```
